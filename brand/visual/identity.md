# Identity

Sora’s visual identity is built around a wordmark and a responsive material / transition system.

The identity should feel warm, grounded, architectural, practical, and quietly innovative. It should make climate-adaptive building materials feel believable and useful in real residential settings.

Sora should not feel like a technology gadget, luxury sustainability brand, sci-fi product, or literal bird-themed identity.

## Core Identity System

Sora’s identity is made from two primary elements:

1. The Sora wordmark
2. The material / transition system

The wordmark provides recognition.  
The material / transition system provides the brand’s distinctive visual behavior.

Together, they should make Sora feel like a responsive architectural material system, not a traditional logo-led consumer product.

## Wordmark Direction

The Sora wordmark should use or reference the Marcellus display direction, with subtle customization to make it feel ownable.

It should feel:

- Warm
- Architectural
- Readable
- Calm
- Refined
- Practical
- Human-centered

The wordmark should not feel decorative, luxury fashion-oriented, historical, overly formal, or overly technological.

A subtle custom detail may appear in the “o,” suggesting Reflect / Absorb behavior, roof zones, environmental transition, or a soft threshold.

The modified “o” should remain legible as part of the wordmark.

Avoid wordmark styles that feel:

- Cold
- Geometric
- Futuristic
- Luxury fashion-oriented
- Overly expressive
- Tech startup-like
- Decorative or ornamental

## Custom Wordmark Detail

The Sora wordmark should include one subtle custom detail.

The preferred custom detail is a modified “o” with a clear but quiet inner curve or division.

This detail may suggest:

- Reflect / Absorb mode shift
- Roof zones
- Environmental transition
- A soft threshold
- Material response
- A curved path of adaptation

The detail should be recognizable but restrained.

It should not become:

- A separate logo symbol
- A diagram
- A bird reference
- A tech icon
- A decorative flourish
- A literal sun/moon symbol
- A complex two-state graphic

The modified “o” should remain legible as part of the wordmark.

## Symbol Use

Sora does not need a separate abstract logo mark in v1.

The brand should rely on the wordmark and material / transition system rather than a standalone icon.

Avoid:

- Bird symbols
- Feather marks
- Wing shapes
- Roof icons
- Thermostat icons
- Sun/moon icons as primary marks
- Generic climate-tech symbols
- Abstract tech marks

If a small identifier is needed, use the modified “o” or a cropped transition/material detail rather than creating a separate symbol.

## Material / Transition System

The material / transition system is a primary identity element.

It may appear through:

- Soft sculptural wave textures
- Fine striated surface detail
- Reflect / Absorb color states
- Organic transition boundaries
- Curved image masks
- Subtle background bands
- Quiet linework
- Divided-light imagery
- Roof-zone diagrams

This system should feel architectural and integrated.

It should not feel like decoration, a graphic overlay, or a digital interface.

## Graphic Identity Behavior

The Sora identity may use transition curves and material-inspired forms as layout devices.

Use them subtly in:

- Image crops
- Section dividers
- Background fields
- Slide layouts
- Campaign panels
- Product diagrams
- Callout areas
- Motion or transition language

These elements should feel calm, sophisticated, and purposeful.

Avoid using the transition language as:

- Loud decoration
- Busy pattern
- Generic wave graphics
- Glowing lines
- Overly literal migration paths
- Decorative swirls
- Dynamic tech effects

## Visual Personality

Sora’s identity should feel:

- Warm
- Grounded
- Calm
- Useful
- Climate-aware
- Material-focused
- Human-centered
- Regionally adaptable
- Quietly innovative
- Architecturally believable

## Identity Guardrails

Do not make Sora feel:

- Sci-fi
- Neon
- Gadget-like
- Luxury-only
- Generic green tech
- Fear-based
- Experimental or fragile
- Literal bird-themed
- Overly polished or artificial
- Disconnected from real homes

## Summary

Sora’s identity should be simple and ownable: a warm Marcellus-based wordmark with one quiet custom detail, supported by a distinctive material and transition system.

The brand should feel like a responsive building material that belongs naturally in everyday residential architecture.
