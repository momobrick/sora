# Campaign Panel Surface

Campaign panels are high-impact static surfaces such as posters, billboards, environmental displays, and brand book hero applications. This is the recommended static touchpoint for Assignment 6.

## Purpose

Use this file for:

- Assignment 6 static execution
- Billboard or poster-style layouts
- Brand campaign panels
- Environmental graphics
- Final project visual world examples

## Surface Role

A campaign panel should communicate the Sora idea in one glance:

**A roof that responds to temperature.**

The visual should make the roof the hero. The copy should reinforce the product behavior without overexplaining it.

## Recommended Composition

The strongest campaign panel composition includes:

- A realistic residential home
- A clearly visible roof plane
- A controlled Reflect / Absorb mode treatment or strong roof material focus
- Clean environmental context
- One primary headline
- Minimal supporting copy

## Static Panel Hierarchy

1. Image
2. Primary headline
3. Optional one-sentence explanation
4. Optional Sora wordmark or product label

## Campaign Panel Typography

Use Marcellus for the primary headline and any major brand statement. Use Inter for supporting copy, product labels, mode labels, and explanatory text.

The campaign panel should use typography sparingly. Marcellus should create a calm, memorable display moment; Inter should keep product information clear and practical.

## Primary Campaign Copy

**A roof that responds to temperature.**

Optional supporting line:

Sora shifts between Reflect Mode and Absorb Mode based on roof surface temperature.

## Visual Treatment Options

### Option A: Divided Lighting Panel

Use the day/dusk split visual system developed for the course.

Purpose:
Show the roof changing between Reflect Mode and Absorb Mode in a single dramatic static image.

Key requirements:

- Preserve the base architecture and landscape.
- Apply a clean, organic dividing curve.
- Left side: daylight / Reflect Mode.
- Right side: dusk / Absorb Mode.
- Do not add a visible line, stroke, glow, or border along the split.
- Keep sky and clouds compositionally continuous.

### Option B: Product Hero Panel

Use a single environmental or architectural image with the Sora roof clearly featured.

Purpose:
Create a cleaner, more believable campaign execution.

Key requirements:

- Roof remains the visual anchor.
- Material is refined and realistic.
- Background supports the climate context without overpowering the product.
- Copy does the explanatory work.

### Option C: Mode Comparison Panel

Use a side-by-side or split-panel composition.

Purpose:
Explain Reflect Mode and Absorb Mode more directly.

Key requirements:

- Both modes feel part of the same material family.
- The comparison is clear without becoming too diagrammatic.
- Labels are restrained and practical.

## Campaign Panel Prompt Scaffold

```text
Create a high-impact static campaign panel for Sora, a responsive roofing material for residential homes. The roof should be the primary visual subject and clearly visible. Communicate the idea: “A roof that responds to temperature.” Show Sora as practical climate adaptation, not futuristic smart-home technology. Preserve architectural realism, natural material quality, and a calm, refined tone. If showing Reflect Mode and Absorb Mode, keep the mode transition controlled and product-accurate. Avoid generic sustainability clichés, solar panels, sci-fi overlays, glowing technology effects, exaggerated weather, and visible AI artifacts.
```

## Divided Lighting Prompt Add-On

```text
Apply a split lighting and roof-color treatment only. Preserve the original house, landscape, camera angle, roof type, roof pattern, windows, doors, sky structure, and environment. Use one broad organic curve spanning from top to bottom of the composition. The left side shows daylight Reflect Mode with realistic shadows. The right side shows dusk Absorb Mode with soft warm window light and reduced shadows. Do not add a white, black, colored, glowing, or outlined line along the division. The same sky and cloud structure should continue across both sides, with only lighting and color temperature changing.
```

## Assignment 6 Translation Notes

The campaign panel is the best static touchpoint because it tests scale. What works in a hero product image may become too detailed or noisy in a billboard-like format.

Watch for:

- Roof texture becoming too busy
- Landscape becoming overly AI-generated
- Split lighting feeling like two unrelated images
- Copy becoming too small or explanatory
- AI adding solar panels, vents, chimneys, or decorative roof patterns

## Recommended Final Project Use

Use the campaign panel as one of the strongest Visual World slides. It can also be reused in the System Logic section to explain how the hero look translates into a public-facing brand application.

## Do

- Keep the image cinematic but believable.
- Make the roof unmistakably central.
- Use one strong headline.
- Simplify environmental detail when needed.
- Preserve product accuracy over visual drama.

## Do Not

- Add excessive copy.
- Make the panel look like generic climate advertising.
- Use apocalyptic weather or fear-based imagery.
- Add sci-fi lighting, interface overlays, or glowing roof effects.
- Let the AI alter the roof type or house structure.

## Quality Check

A successful campaign panel should be legible from a distance. If the viewer only sees a beautiful house and not the responsive roof idea, the panel needs stronger hierarchy.
