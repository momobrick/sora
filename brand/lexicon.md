# Sora Lexicon

Use this file when choosing words, reviewing copy, replacing off-brand language, or building prompts for Sora writing and visual direction.

## Language Philosophy

Sora uses accessible, practical language. The brand should avoid politically loaded, overly technical, overly futuristic, or luxury-coded terminology.

The language should make responsive building materials feel useful, reliable, climate-aware, and relevant to everyday homes and communities.

## Use Carefully

These terms may be appropriate in limited contexts but should be used with care:

### Sustainability

Use only when the context requires it. Sora should not sound like generic sustainability branding or luxury eco-positioning.

Better alternatives: resilience, efficiency, climate adaptation, energy-conscious design, responsive materials.

### Technology

Use only when needed. Sora should not sound like a gadget, software product, or complicated smart-home platform.

Better alternatives: material system, responsive behavior, built-in adaptation, environment-aware performance.

### Innovation

Use when connected to practical value. Do not use innovation as empty praise.

Better alternatives: practical innovation, useful innovation, innovation designed for real homes.

### Future

Use with grounded specificity. Avoid generic “future is here” language.

Better alternatives: future-aware, a necessary evolution in building, designed for changing conditions.

## Better Substitutions

| Instead of | Use |
|---|---|
| green | climate-ready, energy-conscious, environmentally responsive |
| smart | responsive, adaptive, environment-aware |
| luxury sustainable | practical, resilient, broadly attainable |
| high-tech roof | responsive roofing system, climate-adaptive roofing material |
| futuristic | future-aware, forward-looking, built for changing conditions |
| eco-friendly | energy-conscious, climate-adaptive, lower-impact when supported |
| save the planet | help homes and communities adapt |
| climate crisis solution | practical response to changing conditions |
| coming soon | a new way for homes to respond is taking shape |

## Metaphor Language

Sora may lightly draw from migration, seasonal awareness, orientation, movement, shelter, and atmosphere.

Use metaphor as a subtle layer, not a literal theme.

Good metaphor direction:

- guided by changing conditions
- responsive to place and season
- moving with the environment
- oriented toward resilience

Avoid literal bird language unless specifically requested:

- wings
- feathers
- nests
- flock
- soar
- bird mascot language

## Product Language Guardrail

At the brand level, describe Sora as responsive building materials or climate-adaptive building materials. Keep detailed mechanics in the product folder.

When writing about Sora’s first offering, avoid making unsupported claims or describing the product as a coating, gadget, solar panel, decorative add-on, or complete replacement for other building systems.
