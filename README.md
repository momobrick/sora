# Sora

Sora is a fictional responsive roofing brand developed through an AI-aided graphic design process. The project explores how a climate-adaptive building materials company can be defined, governed, and applied across brand strategy, product logic, visual direction, and communication surfaces.

## Core Idea

Buildings should adapt to their environments.

Sora’s initial product focus is a responsive roofing system: a roofing material that shifts between Reflect Mode and Absorb Mode based on roof surface temperature. The brand treats climate adaptation as practical, integrated, and broadly useful for real homes and communities.

## Repository Purpose

This repository is a working brand system for Sora. It is designed to help:

- Keep Sora’s strategy, language, product behavior, and surface applications consistent
- Provide source-of-truth guidance for AI-assisted writing, image generation, and design decisions
- Support course assignments and final brand book development
- Document decisions so new prompts, touchpoints, and brand materials can be created from the same system

## How to Use This Repository

Start with the brand foundation before creating any new Sora output. The foundation defines the core belief, mission, vision, positioning, and role of the brand.

Use the audience, messaging, voice, and lexicon files when writing or reviewing copy. These files help keep Sora clear, warm, grounded, practical, and accessible.

Use the product folder when describing how the responsive roofing system works. Product guidance should be treated as the accuracy layer for all brand, campaign, and surface work.

Use the surfaces folder when translating the brand into specific formats such as campaign panels, social posts, website sections, presentation slides, and product captions.

Use anti-patterns and product accuracy rules as final review filters before approving copy, prompts, imagery, or touchpoint concepts.

## Source-of-Truth Order

When files overlap, use this order:

1. Product accuracy rules override clever language or visual ideas.
2. Brand foundation anchors the meaning, purpose, and positioning.
3. Messaging, voice, and lexicon guide how the brand speaks.
4. Creative direction guides how the brand should feel visually and conceptually.
5. Surface files adapt the system to specific formats without redefining the brand.
6. Anti-patterns identify what to remove, revise, or avoid.

## Key Guardrails

Across all Sora work:

- Sora is the roofing material itself, not a coating, paint, film, or add-on finish.
- The roof responds to roof surface temperature, not general weather, time of day, season, forecast, or user mood.
- Reflect Mode and Absorb Mode should remain visually distinct while belonging to one material system.
- Sora should feel practical, warm, climate-aware, and broadly attainable.
- Avoid unsupported performance claims, fear-based climate messaging, luxury-only framing, generic sustainability clichés, and sci-fi technology aesthetics.

## Project Status

This is an evolving academic and conceptual brand system. It is not a commercial product specification, engineering document, or validated performance claim. Files should continue to evolve as the Sora brand system, visual world, product logic, and assignment outputs develop.
